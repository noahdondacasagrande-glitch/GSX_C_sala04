

Estrutura Básica de um Programa em C
Todo programa em C tem como ponto de entrada a função main(). A diretiva #include importa bibliotecas e return 0 indica encerramento com sucesso.
#include <stdio.h>
int main() {
    printf("Olá, mundo!");
    return 0;
}
Tipos de Dados
Os tipos básicos da linguagem C são: int (inteiros), float (reais de precisão simples), double (reais de maior precisão) e char (caracteres individuais). A escolha correta do tipo otimiza o uso de memória e garante a precisão dos cálculos.
c
int idade = 20;   float altura = 1.75;   char letra = 'A';
Variáveis e Constantes
Variáveis armazenam valores que podem ser alterados durante a execução. Constantes, declaradas com const, possuem valor fixo e tornam o código mais seguro e fácil de manter.
int numero = 10;   numero = 20;
const float PI = 3.14159;
Operadores
Aritméticos: + - * / % — usados em cálculos matemáticos. Relacionais: == != > < >= <= — usados para comparar valores. Lógicos: && (E), || (OU), ! (NÃO) — usados para combinar condições.
if (idade >= 18 && idade < 60) { ... }
Estruturas de Controle
O if/else executa blocos conforme uma condição. O switch é ideal quando há várias opções de escolha.
if (idade >= 18) { printf("Maior de idade"); }
else             { printf("Menor de idade"); }
switch(opcao) {
    case 1: printf("Cadastrar"); break;
    case 2: printf("Consultar"); break;
    default: printf("Opção inválida");
}
Estruturas de Repetição
for — usado quando o número de repetições é conhecido.
while — repete enquanto a condição for verdadeira.
do-while — garante a execução do bloco ao menos uma vez.
for (int i = 0; i < 5; i++) { printf("%d\n", i); }
Funções
Permitem dividir o programa em partes reutilizáveis, reduzindo repetição e melhorando a legibilidade do código.
int soma(int a, int b) { return a + b; }
int resultado = soma(5, 3);
Vetores e Matrizes
Vetores armazenam múltiplos valores do mesmo tipo em posições consecutivas de memória. Matrizes funcionam como tabelas com linhas e colunas.
int numeros[5] = {1, 2, 3, 4, 5};
printf("%d", numeros[0]);
int matriz[3][3];
Ponteiros
Armazenam endereços de memória em vez de valores. São essenciais para alocação dinâmica, manipulação de estruturas complexas e desenvolvimento de sistemas de baixo nível.
int x = 10;
int *p = &x;
printf("%d", *p);  // exibe 10
Entrada e Saída de Dados
A interação com o usuário é feita pelas funções printf() (saída) e scanf() (entrada), disponíveis na biblioteca stdio.h.
printf("Bem-vindo!");
int idade;
scanf("%d", &idade);
